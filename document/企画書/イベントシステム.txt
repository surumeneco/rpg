　→たまにアストラルウェポンをドロップ

・アストラルウェポン
　→ランダムな性能の武器
　　→エンドコンテンツ的な感じ
　→性能はコスト式にするかも
武器種：全種から
攻撃力：3000〜5000
属性値：1000〜3000
状態異常値：1000〜3000
クリティカル率：50%の確率で+5%を10回
　→for(10回) { if(50%) { +5% } }
ファンブル率：40%の確率で+5%を10回
　→for(10回) { if(40%) { +5% } }
スキルスロット：+1個当たり75%を7階層
　→for(7回) { if(100%-75%) { break; } +1個 }
